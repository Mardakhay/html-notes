What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of a page using a series of elements represented by tags. Browsers read HTML and render it visually for users.

üèóÔ∏è

Structure

HTML defines the skeleton of every webpage ‚Äî headings, paragraphs, images, links, and more.

üè∑Ô∏è

Tags

Elements are written using <tags>. Most have an opening <tag> and closing </tag>.

‚öôÔ∏è

Attributes

Tags can carry attributes like class, id, href, src that add extra information.

üåê

HTML5

The latest standard. Adds semantic tags, native audio/video, canvas, and modern form inputs.

‚ôø

Accessibility

Semantic HTML improves screen reader support and makes pages more accessible to everyone.

üîç

SEO

Proper HTML structure helps search engines understand and rank your content better.

üí° HTML, CSS & JavaScript

HTML provides structure, CSS adds style, and JavaScript adds behaviour. They are the three core technologies of every webpage.

Nesting Rules
Elements can be nested inside each other. The inner element must be closed before the outer one.
HTML
<p>This is <strong>bold</strong> and <em>italic</em> text.</p>
‚ö†Ô∏è Incorrect Nesting

Block vs Inline Elements
Block elements start on a new line and take up the full width available.
<div><p><h1>‚Äì<h6> <ul><ol><li> <section><article><header> <footer><main><nav> <table><form><blockquote>
Inline elements sit within the flow of text and only take up as much width as their content.
<span><a><strong> <em><img><br> <code><label><input> <button><abbr><sup>

Document Structure

Every HTML document follows a consistent boilerplate structure that tells the browser how to interpret and display it.

The Boilerplate
HTML5

<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Webpage</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <h1>Welcome to My Website</h1>
    <p>This is some content.</p>
    <script src="app.js"></script>
  </body>
</html>
üìã

<!DOCTYPE html>

Declares the document type. Always the very first line ‚Äî tells browsers to use HTML5 mode.

üåê

<html lang="en">

Root element of the page. The lang attribute helps browsers and screen readers with language.

üß†

<head>

Contains metadata: title, charset, viewport, linked CSS/JS. Not visible to users.

üëÅÔ∏è

<body>

Everything inside here is rendered and shown to users in the browser window.

‚úÖ Best Practice

Always include <meta charset="UTF-8"> to support all characters and <meta name="viewport"> for proper mobile rendering.

Links & Images

Anchor tags create hyperlinks to pages, files, or locations. Images embed visual media with fallback text.

Anchor Tag <a>
HTML
<!-- External link (opens new tab) -->
<a href="https://example.com" target="_blank" rel="noopener">External link</a>

<!-- Internal link -->
<a href="/about">Internal link</a>

<!-- Anchor / jump link -->
<a href="#section-id">Jump to section</a>

<!-- Email & phone -->
<a href="mailto:user@example.com">Send email</a>
<a href="tel:+1234567890">Call us</a>
href ‚Äî destination URL target="_blank" ‚Äî new tab rel="noopener" ‚Äî security download ‚Äî force download title ‚Äî tooltip text
‚ö†Ô∏è Security

Always add rel="noopener noreferrer" when using target="_blank" to prevent the new page from accessing your page via window.opener.

Image Tag <img>
HTML
<img src="photo.jpg" alt="A mountain landscape"
     width="600" height="400" loading="lazy">

<!-- With caption -->
<figure>
  <img src="sunset.jpg" alt="Sunset over sea">
  <figcaption>Sunset over the ocean.</figcaption>
</figure>
src ‚Äî image path/URL alt ‚Äî fallback text width / height loading="lazy" srcset ‚Äî responsive images
‚úÖ Always use alt text

The alt attribute is essential for accessibility and SEO. Describe what the image shows. Use alt="" for purely decorative images.

Head & Meta Tags

The <head> section contains invisible metadata ‚Äî information for browsers, search engines, and social media scrapers.

  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Page Title</title>

  <meta name="author" content="Your Name">

  <meta property="og:image" content="/preview.jpg">

  <!-- Favicon -->
  <link rel="icon" href="/favicon.ico">

  <!-- Stylesheets and fonts -->
  <link rel="stylesheet" href="style.css">
</head>
